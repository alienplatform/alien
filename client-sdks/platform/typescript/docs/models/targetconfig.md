# TargetConfig

Deployment configuration

Configuration for how to perform the deployment.
Note: Credentials (ClientConfig) are passed separately to step() function.

## Example Usage

```typescript
import { TargetConfig } from "@alienplatform/platform-api/models";

let value: TargetConfig = {
  environmentVariables: {
    createdAt: "1707747302593",
    hash: "<value>",
    variables: [],
  },
};
```
