# SyncReconcileResponseComputeBackend

Compute backend for Container and Function resources.

Determines how compute workloads are orchestrated on cloud platforms.
When None, the platform default is used (Horizon for cloud platforms).

## Example Usage

```typescript
import { SyncReconcileResponseComputeBackend } from "@aliendotdev/platform-api/models";

let value: SyncReconcileResponseComputeBackend = {
  clusters: {
    "key": {
      clusterId: "<id>",
      managementToken: "<value>",
    },
  },
  horizondDownloadBaseUrl: "https://shiny-partridge.info/",
  url: "https://neighboring-technician.name/",
  type: "horizon",
};
```
