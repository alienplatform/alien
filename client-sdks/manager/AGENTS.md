# Manager SDK

Auto-generated API clients for alien-manager (OSS control plane).

## Contents

- `rust/` — `alien-manager-api` crate, generated by progenitor from `openapi.json`
- `typescript/` — `@aliendotdev/manager-api` package, generated by Speakeasy
- `openapi.json` — OpenAPI 3.1 spec (source of truth for generation)
- `openapi-3.0.json` — Downgraded spec for tools that need 3.0
- `scripts/` — Generation and conversion scripts

## Regenerating

```bash
# From repo root
pnpm run generate:api-rust-sdk   # Rust only
pnpm run generate:api            # All SDKs
```

The OpenAPI spec is generated from alien-manager's Axum routes (utoipa annotations).

## Inspecting Rust Types

```bash
cargo build -p alien-manager-api
CODEGEN=$(find target/debug/build -path "*alien-manager-api*/out/codegen.rs" -type f | xargs ls -t 2>/dev/null | head -1)
grep -A 20 "pub struct Deployment" $CODEGEN
```

## Don't

- Don't edit generated code manually — it will be overwritten
- Don't use "agent" or "server" in the API — use "deployment" and "manager"
